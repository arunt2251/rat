# Project Name
RAT (Remote Access Tool) in Bash

## Description
This project is a simple Remote Access Tool (RAT) implemented in C and converted to Bash for educational purposes. It demonstrates basic client-server communication and remote command execution. **This project is intended for learning and ethical hacking practice only. Do NOT use it for illegal purposes.**

## Features
- Connects a client to a server
- Execute basic shell commands remotely
- Transfer files between client and server (if implemented)
- Simple logging of client activity

## Requirements
- Linux environment
- Bash shell
- Network access between client and server
